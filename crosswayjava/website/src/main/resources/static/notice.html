购买须知

【下单后信息不可修改】

用户在下单时填写的收货信息，在您成功购买后，将作为您的最终收货信息，一经提交，不可更改。信息包含姓名、电话、收货地址、发票抬头、型号、颜色等。

【销售区域限制】

由于物流配送原因，部分偏远地区暂时无法服务，请您购买前先咨询客服确认所在地址是否可以配送。

我们将每天对订单信息进行筛选，无法送达的订单，我们将主动与您联系，并在7个工作日内进行退款。请您保持电话畅通。

【关于发货】

下单后您的商品会在三个工作日内从工厂发出。发货后正常情况下在七个工作日内可送到。如需上楼，根据情况可能收取上楼费。

您可以登录（http://www.klsw.com/Order/Index），进入“我的订单”了解订单信息。

【关于发票】

克洛斯威默认开具的发票是普通专用发票。发票抬头以您订单填写为准，订单一经提交，信息不可更改。发票为额外寄出，您也可以联系客服选择电子发票或增值税发票（需提供税务资料）。

若团体或公司购买，暂时无法为您办理增值税专用发票，请您谨慎下单。

【物流进度查询】

您可以登录克洛斯威官网（http://www.klsw.com/），点击订单查询，查询您的物流进度。

购物指南

克洛斯威网站购物流程 1.新用户注册 2.挑选商品 3.确认下单 4、订单付款 5.订单支付 6.验货签收

1.请登录克洛斯威网站（http://www.klsw.com/）

如果您没有克洛斯威帐号，请点击此网页右上方“注册”按钮，并根据页面引导完成注册；

第一步 进入克洛斯威网站，点击“注册”按钮进行注册。

第二步 进入注册页面，根据相关的提示内容填写注册信息。

第三步 账号注册完成后登录克洛斯威网站，挑选心仪的商品。点击“加入购物车”。如果还需要添加其他商品，可点击“继续购物”，挑选更多商品。

第四步 填写购物信息

点击“下一步”，填写收货人、地址、手机号等信息、并选择支付方式、配送方式、发票类型等。订单提交后无法修改，请您付款前仔细核实。

第五步 订单支付

确认购物信息及“商品清单”无误后，点击“去支付”，依据页面的提示，选择您熟悉的支付方式，并完成支付。

第六步 验货签收

收到货后，请当面验收，如有任何问题，您可以选择拒收，或者拨打400-808-0218咨询客服。如果配送员不配合送货、验收等，也可拨打400电话进行投诉。

支付方式

克洛斯威官方网站提供支付宝、微信等多种支付方式，同时支持国内二十多家主流银行机构的储蓄卡、信用卡的网上付款。暂时不支持货到付款。请亲们购买时留意，避免影响您的正常购物。

配送方式

克洛斯威智能钢琴采用第三方快递配送（确认合作物流公司后，明示配送的物流公司名称）。用户可以在“我的订单”里查询物流信息，如有其他疑问请咨询克洛斯威官方客服人员。

售后服务须知

1. 克洛斯威的基于用心服务，专注用户满意的服务理念，严格执行国家三包规定。三包有效期内，正常使用情况下，因产品本身质量问题引起的维修（含维修费用）、换货、退货，属免费三包服务范围。三包政策适用于中华人民共和国大陆地区（不含香港、澳门、台湾地区）。


2. 我们会为所有用户提供发票及保修卡作为商品的质保凭证，请您妥善保留以便作为质保之用。如不能出示有效发票则不能享受7 天免费退换货服务，但您还可以继续享受到本公司所提供的免费保修服务。


3. 购物后15日内享有退换货服务的用户，在办理服务时，用户必须提供对应商品的发票及保修卡，并保证商品的完整，包括原来的包装箱、说明书及商品附带着的所有附件，且不影响商品的再次销售。如果某部分丢失或损坏，用户需支付相应的材料费用，否则我们无法为您办理相应的退换货服务。


4. 克洛斯威所销售的产品，在实施细则上，如保修操作、售后服务、技术支持等方面为您提供的服务及所有保修规定均以国家《三包规定》为准则，如本说明有和《三包规定》相冲突之处，以《三包规定》为准。


5.送货上门：您收货时请务必开箱验货，若被要求收取与您约定之外的其他费用，或验货时有外包装及商品问题，请第一时间联系我司客服。

特别情况须知

下列情况不在国家相关《三包条例》之列

1. 超过三包有效期；

2. 未按产品正常使用的要求使用、维护、保管而造成损坏；

3. 非承担三包的修理者（即非厂商指定或特约维修点）拆动造成损坏；

4. 没有有效三包凭证及有效发票；

5. 擅自涂改三包凭证；

6. 三包凭证上的产品型号或编号与商品实物不相符合；

7. 因不可抗力（如：地震、水灾、火灾）原因造成损坏；

8. 非产品本身质量问题，如：软件兼容性问题，对于颜色，外观，形状不满意等；

特别提示：建议您将商品外包装在收货之日起保留30天以上。如包装破损或丢失，会影响您商品退换货的正常办理。
